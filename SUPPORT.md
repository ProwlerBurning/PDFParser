# Support

## How to get help

Support is provided through **GitHub Issues**. Search existing issues first, then
open a new one using the appropriate template.

## Expectations

- There is **no SLA** — responses are best-effort.
- This project is **community-maintained**.
- It is **not affiliated** with any bank, e-wallet provider, financial
  institution, or any employer.

## Privacy when asking for help

- **Do not upload real statements** or any real financial data.
- Use **synthetic or redacted examples only**.
- Scrub logs of account numbers, wallet IDs, card numbers, and other personal
  identifiers before posting.

See [SECURITY.md](SECURITY.md) for how to report sensitive issues privately and
[docs/PRIVACY_MODEL.md](docs/PRIVACY_MODEL.md) for the project's privacy design.
